Node Doctor Management API

Project Setup

1. Xampp server
   Run the xampp server
   Import DB

2. Install dependencies:
   npm install

3. Run the project in development mode:
   npm run dev

   The server will start on the port defined in your .env file (default: 5000).

Stakeholders

Doctors
- Can signup
- Can add their complete details (certifications, specialization, etc.)

Admin
- Can get all doctors
- Can get doctor by ID
- Can reject/delete doctor
